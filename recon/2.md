**Ping** <br><br>
**Ping is an active reconnaissance tool that sends ICMP requests to check if a target system is online and reachable based on its reply.**
**Questions:** <br>
**1) Which option would you use to set the size of the data carried by the ICMP echo request?** <br>
**-> -s**
<br><br>
**2) What is the size of the ICMP header in bytes?** <br>
**-> 8**
<br><br>
**3) Does MS Windows Firewall block ping by default? (Y/N)** <br>
**-> Y**
<br><br>
**4) Deploy the VM for this task and using the AttackBox terminal, issue the command ping -c 10 MACHINE_IP. How many ping replies did you get back?**
**-> 10**
