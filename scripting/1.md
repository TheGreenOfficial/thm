**Mathemetical Operators** <br>
*Arithmetic Operators*

| Operator        | Syntax | Example         |
|----------------|--------|-----------------|
| Addition       | `+`    | `1 + 1 = 2`     |
| Subtraction    | `-`    | `5 - 1 = 4`     |
| Multiplication | `*`    | `10 * 10 = 100` |
| Division       | `/`    | `10 / 2 = 5`    |

**Loops** <br>
**In programming, loops allow programs to iterate and perform actions a number of times. There are two types of loops, for and while loops.** <br>
```
# eg while loop:
i = 1
while i <= 10:
     print(i)
     i = i + 1

#eg for loop:
websites = ["facebook.com", "google.com", "amazon.com"]
for site in websites:
     print(site)

#eg second for loop:
for i in range(5):
     print(i)
```

**Introduction to Functions** <br>
**As programs start to get bigger and more complex, some of your code will be repetitive, writing the same code to do the same calculations, and this is where functions come in. A function is a block of code that can be called at different places in your program.** <br>
```
# eg:
def sayHello(name):
     print("Hello " + name + "! Nice to meet you.")

sayHello("ben") # Output is: Hello ben! Nice to meet you

# another example:
def calcCost(item):
     if(item == "sweets"):
          return 3.99
     elif (item == "oranges"):
          return 1.99
     else:
          return 0.99

spent = 10
spent = spent + calcCost("sweets")
print("You have spent:" + str(spent))
```

**Files** <br>
**In Python, you can read and write from files. We've seen that in cyber security, it's common to write a script and import or export it from a file; whether that be as a way to store the output of your script or to import a list of 100's of websites from a file to enumerate. Let's dive straight into an example:** <br>
```
# eg one: 
f = open("file_name", "r")
print(f.read())

# eg two:
f = open("demofile1.txt", "a") # Append to an existing file
f.write("The file will include more text..")
f.close()

# eg three:
f = open("demofile2.txt", "w") # Creating and writing to a new file
f.write("demofile2 file created, with this content in!")
f.close()
```

**Imports** <br>
**In Python, we can import libraries, which are a collection of files that contain functions. Think of importing a library as importing functions you can use that have been already written for you. For example, there is a "date" library that gives you access to hundreds of different functions for anything date and time-related.** <br>
